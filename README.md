## Building
- Linux
  ```bash
  mkdir build
  cd build
  cmake ..
  cmake --build .
  ```

- Windows
  1. Building using CMake tools in Visual Studio Community
  2. Building using CMake toolkit in Visual Studio Code
  3. Building using Powershell just like in Linux
